### 1.2 문제점 파악

#### 정보 과부하
- **탭 2 (Supabase 필터)**에 9가지 필터가 한 번에 나열되어 압도적
- 사용자가 필요한 필터를 찾기 어려움
- 스크롤이 길어져 모바일 환경에서 특히 불편

#### 기능 분산
- 검색 기능이 필터 탭 중간에 위치 (가장 많이 사용하는 기능인데 접근성 낮음)
- 제품 선택과 필터가 별도 탭으로 분리되어 있어 워크플로우가 끊김

#### 시각적 구분 미흡
- 모든 필터가 같은 레벨로 표시되어 중요도 구분 없음
- 활성 필터 요약이 있지만 시각적으로 눈에 띄지 않음

#### 필터 히스토리 기능 미노출
- 필터 히스토리 되돌리기 버튼이 사이드바 최상단에 있어서 눈에 띄지 않음
- 필터 저장/복원 기능이 있지만 활용도가 낮음

## 2. 개선 제안

### 2.1 탭 기반 UI 재구성 방안

#### 새로운 3개 탭 구조

```
🔍 검색 & 필터  |  📊 통계 & 인사이트  |  ⚙️ 설정
```

**탭 1: 🔍 검색 & 필터** (핵심 기능 통합)
- 검색창 (최상단 배치)
- 제품 선택 (필수)
- 기본 필터 (expander) - 카테고리, 브랜드, 신뢰도
- 고급 필터 (expander) - 가격, 평점, 리뷰 수, 날짜, 언어
- 활성 필터 요약 카드
- 필터 관리 버튼 (초기화, 저장, 되돌리기)

**탭 2: 📊 통계 & 인사이트** (기존 탭 3 확장)
- 실시간 통계 (전체 제품, 리뷰, 평균 가격)
- 필터 적용 결과 미리보기
- 브랜드별/카테고리별 통계
- 평점 분포

**탭 3: ⚙️ 설정** (기존 탭 4 + 추가 기능)
- 필터 프리셋 관리
- 표시 옵션 (테마, 차트 높이 등)
- 안내 (신뢰도 등급, 분석 기준, 사용 팁)

### 2.3 UX 개선 포인트

#### Quick Win (즉시 적용 가능)

1. **검색창 최상단 배치**
   - 가장 많이 사용하는 기능을 가장 접근하기 쉬운 위치로
   - 사이드바 열자마자 바로 보이도록

2. **필터 그룹화**
   - 기본 필터 (카테고리, 브랜드, 신뢰도) vs 고급 필터 (가격, 평점, 리뷰 수, 날짜, 언어)
   - expander로 접었다 펼 수 있어 정보 과부하 방지

3. **활성 필터 시각화**
   - 현재 적용된 필터를 카드 형태로 시각적 강조
   - 필터 적용 상태를 한눈에 파악 가능

4. **필터 관리 버튼 개선**
   - 초기화/저장/되돌리기 버튼을 눈에 띄게 배치
   - 아이콘 + 텍스트 조합으로 직관성 향상

#### Medium Win (약간의 코드 수정 필요)

5. **필터 프리셋 기능**
   - 자주 사용하는 필터 조합을 프리셋으로 저장
   - 예: "고신뢰도 제품만", "가성비 우선", "최신 리뷰만" 등

6. **필터 모드 전환**
   - st.radio를 활용하여 "빠른 필터" vs "상세 필터" 전환
   - 초보자는 빠른 필터, 고급 사용자는 상세 필터 사용

7. **필터 적용 상태 색상 구분**
   - 활성화된 필터는 파란색 테두리
   - 비활성화된 필터는 회색

#### Long-term (추후 개선)

8. **실시간 결과 미리보기**
   - 필터 조정 시 적용될 제품 수를 실시간으로 표시
   - 예: "이 조건으로 3개 제품이 표시됩니다"

9. **AI 필터 추천**
   - 사용자 패턴 분석하여 필터 조합 추천
   - 예: "이 필터 조합을 자주 사용하시네요. 저장하시겠어요?"

### 3.2 After: 개선 코드

```python
# ========== 사이드바: 개선된 3개 탭 구조 ==========
with st.sidebar:
    # Supabase 연결 상태
    if USE_SUPABASE:
        st.success("✅ Supabase 연동 활성화")
    else:
        st.warning("⚠️ 목업 데이터 사용 중")

    # 3개 탭 (기능 통합)
    sidebar_tab1, sidebar_tab2, sidebar_tab3 = st.tabs([
        "🔍 검색 & 필터",
        "📊 통계 & 인사이트",
        "⚙️ 설정"
    ])

    # ========== 탭 1: 검색 & 필터 ==========
    with sidebar_tab1:
        # 1. 검색창 (최상단 배치)
        st.markdown("### 🔎 검색")
        search_query = st.text_input(
            "제품 검색",
            placeholder="예: NOW Foods, Lutein...",
            key="search_query",
            label_visibility="collapsed"
        )

        st.markdown("---")

        # 2. 제품 선택 (필수)
        st.markdown("### 📦 제품 선택")
        selected_labels = st.multiselect(
            "분석할 제품을 선택하세요",
            options=list(product_options.keys()),
            default=list(product_options.keys())[:3],
            key="product_select"
        )

        # 빠른 선택 버튼
        col_q1, col_q2 = st.columns(2)
        with col_q1:
            if st.button("상위 3개", use_container_width=True, key="quick_top3"):
                st.session_state.product_select = list(product_options.keys())[:3]
                st.rerun()
        with col_q2:
            if st.button("전체 선택", use_container_width=True, key="quick_all"):
                st.session_state.product_select = list(product_options.keys())
                st.rerun()

        st.markdown("---")

        # 3. 기본 필터 (기본 펼침)
        with st.expander("📂 기본 필터", expanded=True):
            # 카테고리
            if categories:
                category_filter = st.multiselect(
                    "📂 카테고리",
                    options=categories,
                    default=categories,
                    key="category_filter"
                )
            else:
                category_filter = []

            # 브랜드
            if brands:
                brand_filter = st.multiselect(
                    "🏷️ 브랜드",
                    options=brands,
                    default=brands,
                    key="brand_filter"
                )
            else:
                brand_filter = []

            # 신뢰도
            trust_filter = st.multiselect(
                "🎯 신뢰도 등급",
                options=["HIGH", "MEDIUM", "LOW"],
                default=["HIGH", "MEDIUM", "LOW"],
                key="trust_filter"
            )

        # 4. 고급 필터 (기본 접힘)
        with st.expander("⚙️ 고급 필터", expanded=False):
            # 가격 범위
            if all_products_list:
                prices = [p.get("price", 0) for p in all_products_list if p.get("price") and p.get("price") > 0]
                if prices:
                    price_range = st.slider(
                        "💰 가격 범위 ($)",
                        min_value=float(min(prices)),
                        max_value=float(max(prices)),
                        value=(float(min(prices)), float(max(prices))),
                        key="price_range"
                    )

            # 평점 범위
            if all_products_list:
                ratings = [p.get("rating_avg", 0) for p in all_products_list if p.get("rating_avg") and p.get("rating_avg") > 0]
                if ratings:
                    rating_range = st.slider(
                        "⭐ 평점 범위",
                        min_value=float(min(ratings)),
                        max_value=float(max(ratings)),
                        value=(float(min(ratings)), float(max(ratings))),
                        step=0.1,
                        key="rating_range"
                    )

            # 리뷰 수
            if all_products_list:
                review_counts = [p.get("rating_count", 0) for p in all_products_list if p.get("rating_count")]
                if review_counts:
                    review_count_range = st.slider(
                        "💬 리뷰 수 범위",
                        min_value=int(min(review_counts)),
                        max_value=int(max(review_counts)),
                        value=(int(min(review_counts)), int(max(review_counts))),
                        key="review_count_range"
                    )

            # 날짜 필터
            st.markdown("**📅 리뷰 날짜 범위**")
            col_date1, col_date2 = st.columns(2)
            with col_date1:
                start_date = st.date_input("시작일", value=None, key="review_start_date")
            with col_date2:
                end_date = st.date_input("종료일", value=None, key="review_end_date")

            # 언어 필터
            language_filter = st.multiselect(
                "🌐 리뷰 언어",
                options=["ko", "en", "all"],
                default=["all"],
                key="language_filter"
            )

        st.markdown("---")

        # 5. 활성 필터 요약 (시각적 강조)
        active_filters = get_active_filters_summary(filters_dict, all_products_list)
        if active_filters:
            st.markdown("""
            <div style="background: #FEF3C7; border-left: 4px solid #F59E0B; padding: 1rem; border-radius: 6px; margin-bottom: 1rem;">
                <p style="margin: 0; font-weight: 600; color: #92400E;">
                    🔍 활성 필터: {0}개
                </p>
            </div>
            """.format(len(active_filters)), unsafe_allow_html=True)

            for f in active_filters:
                st.caption(f"  • {f}")

        st.markdown("---")

        # 6. 필터 관리 버튼 (3개 버튼)
        col_btn1, col_btn2, col_btn3 = st.columns(3)
        with col_btn1:
            if st.button("🔄", help="초기화", use_container_width=True, key="reset_filters"):
                safe_categories = categories if (categories is not None and isinstance(categories, list)) else []
                safe_brands = brands if (brands is not None and isinstance(brands, list)) else []
                safe_products = all_products_list if (all_products_list is not None and isinstance(all_products_list, list)) else []
                reset_all_filters(safe_products, safe_categories, safe_brands)
                st.rerun()

        with col_btn2:
            if st.button("💾", help="저장", use_container_width=True, key="save_filters"):
                current_filters = {
                    'category_filter': st.session_state.get('category_filter', []),
                    'brand_filter': st.session_state.get('brand_filter', []),
                    'price_range': st.session_state.get('price_range', None),
                    'rating_range': st.session_state.get('rating_range', None),
                    'review_count_range': st.session_state.get('review_count_range', None),
                    'trust_filter': st.session_state.get('trust_filter', []),
                    'search_query': st.session_state.get('search_query', ''),
                    'review_start_date': st.session_state.get('review_start_date', None),
                    'review_end_date': st.session_state.get('review_end_date', None),
                    'language_filter': st.session_state.get('language_filter', ['all'])
                }
                save_filter_state_to_history(current_filters)
                st.success("저장 완료!", icon="✅")

        with col_btn3:
            if 'filter_history' in st.session_state and len(st.session_state.filter_history) > 0:
                if st.button("↩️", help="되돌리기", use_container_width=True, key="undo_filters"):
                    previous_state = restore_filter_state_from_history()
                    if previous_state:
                        # 필터 상태 복원
                        for key, value in previous_state.items():
                            st.session_state[key] = value
                        st.rerun()

    # ========== 탭 2: 통계 & 인사이트 ==========
    with sidebar_tab2:
        st.markdown("### 📊 실시간 통계")

        try:
            stats = get_cached_statistics()

            # 전체 통계
            col_s1, col_s2 = st.columns(2)
            with col_s1:
                st.metric("전체 제품", f"{stats.get('total_products', 0)}개", delta=None, delta_color="off")
            with col_s2:
                st.metric("전체 리뷰", f"{stats.get('total_reviews', 0)}개", delta=None, delta_color="off")

            st.metric("평균 가격", f"${stats.get('avg_price', 0):.2f}", delta=None, delta_color="off")

            st.markdown("---")

            # 필터 적용 결과 미리보기
            st.markdown("### 🎯 필터 적용 결과")
            if 'selected_data' in locals() and selected_data:
                filtered_count = len(selected_data)
                filtered_reviews = sum(len(d.get("reviews", [])) for d in selected_data)

                st.info(f"**{filtered_count}개 제품** ({filtered_reviews}개 리뷰)")
            else:
                st.caption("필터를 적용하여 결과를 확인하세요")

            st.markdown("---")

            # 브랜드별 통계
            st.markdown("### 🏷️ 브랜드별 통계")
            brand_stats = stats.get('brands', {})
            if brand_stats:
                for brand, data in sorted(brand_stats.items(), key=lambda x: x[1]['count'], reverse=True)[:5]:
                    avg_rating = data['total_rating'] / data['count'] if data['count'] > 0 else 0
                    st.markdown(f"**{brand}**")
                    st.caption(f"제품: {data['count']}개 | 평점: {avg_rating:.1f} | 리뷰: {data['total_reviews']}개")

            st.markdown("---")

            # 평점 분포
            st.markdown("### ⭐ 평점 분포")
            rating_dist = stats.get('rating_distribution', {})
            if rating_dist:
                total_ratings = sum(rating_dist.values())
                for rating in [5, 4, 3, 2, 1]:
                    count = rating_dist.get(rating, 0)
                    percentage = (count / total_ratings * 100) if total_ratings > 0 else 0
                    st.progress(percentage / 100, text=f"{rating}점: {count}개 ({percentage:.1f}%)")

        except Exception as e:
            st.error(f"통계 로드 실패: {e}")

    # ========== 탭 3: 설정 ==========
    with sidebar_tab3:
        st.markdown("### ⚙️ 설정")

        # 필터 프리셋 (추가 기능)
        st.markdown("#### 💾 필터 프리셋")
        preset = st.selectbox(
            "빠른 필터 적용",
            ["선택 안함", "고신뢰도 제품만", "가성비 우선", "최신 리뷰만"],
            key="filter_preset"
        )

        if preset == "고신뢰도 제품만":
            if st.button("적용", key="apply_preset_high_trust"):
                st.session_state.trust_filter = ["HIGH"]
                st.success("고신뢰도 필터 적용!")
                st.rerun()
        elif preset == "가성비 우선":
            if st.button("적용", key="apply_preset_value"):
                # 가격 낮은순 + 신뢰도 HIGH/MEDIUM
                st.session_state.trust_filter = ["HIGH", "MEDIUM"]
                if all_products_list:
                    prices = [p.get("price", 0) for p in all_products_list if p.get("price") and p.get("price") > 0]
                    if prices:
                        st.session_state.price_range = (float(min(prices)), float(min(prices)) * 1.5)
                st.success("가성비 필터 적용!")
                st.rerun()

        st.markdown("---")

        # 표시 옵션
        st.markdown("#### 🎨 표시 옵션")
        show_chart_analysis = st.checkbox("AI 차트 분석 활성화", value=True, key="show_ai_analysis")
        chart_height = st.slider("차트 높이", 300, 800, 400, 50, key="chart_height")

        st.markdown("---")

        # 안내
        st.markdown("### ℹ️ 안내")

        with st.expander("🎯 신뢰도 등급", expanded=False):
            st.markdown("""
            - **HIGH (70점 이상)**: 신뢰할 수 있는 제품
            - **MEDIUM (50-70점)**: 보통 수준
            - **LOW (50점 미만)**: 주의 필요
            """)

        with st.expander("📊 분석 기준", expanded=False):
            st.markdown("""
            1. 인증 구매 비율
            2. 재구매율
            3. 장기 사용 비율
            4. 평점 분포 적절성
            5. 리뷰 길이
            6. 시간 분포 자연성
            7. 광고성 리뷰 탐지
            8. 리뷰어 다양성
            """)

        with st.expander("💡 사용 팁", expanded=False):
            st.markdown("""
            - **검색창**에서 제품명이나 브랜드를 검색하세요
            - **기본 필터**로 빠르게 필터링하세요
            - **고급 필터**로 세밀한 조건을 설정하세요
            - **프리셋**으로 자주 사용하는 필터 조합을 적용하세요
            """)
```
